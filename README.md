## 校园网上超市
### 项目地址：http://lhbpi.free.idcfengye.com/shop_c
### 部署于树莓派服务器，通过Ngrok形成内网穿透（速度较慢）
### <div style="color:red">页面显示”Tunnel ** not found“基本是服务器暂时关闭，主要7:00~23:00开启</div>
    主要功能：
    1.提供商品的购买，同时可收藏商品、添加商品到购物车，支持通过商品名称进行搜索；
    2.商品下单后可进行取消订单，已支付订单可申请退款，退款后续操作
    3.于后台进行，确认后金额自动返还；
    4.已支付订单亦可立即确认收货、完成等操作；
    5.已完成订单可进行评分与评论；
    6.订单完成前可申请退货、退货退款，后续确认操作由后台提供；
    7.提供金额消费、返还、充值记录显示；
